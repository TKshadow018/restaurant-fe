Menu Item Modal - Volume Selection Test

Test Case 1: Multiple Volumes (Pizza)

Item: Chicken Kebab Pizza
Available Volumes:
Small - 69 SEK
Medium - 119 SEK
Large - 139 SEK

Expected Behavior: Should show volume selector with 3 options, price updates based on selection

Test Case 2: Single Volume (Side Item)

Item: Wings (5 pcs)
Available Volumes:
Normal - 39 SEK

Test Case 3: Volume Names Translation

English: Small, Medium, Large, Normal
Swedish: Liten, Mellan, Stor, Normal

Expected Behavior: Volume names should change based on selected language

  • ✅ Volume/Size selector appears when multiple sizes are available

  • ✅ Volume names are localized (Swedish/English)
  • ✅ Selected volume is included in cart item
  • ✅ Add to Cart button shows selected size
  • ✅ Volume selection resets to first option when modal opens
  • ✅ Handles both legacy and new price structures
