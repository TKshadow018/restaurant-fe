Restaurant Favicon Generator

Instructions to create favicon files:

  1. Open the dining.png file in any vector graphics editor (Inkscape, Adobe Illustrator, or online tools like Photopea)
  2. Export/Save as PNG files with these sizes:
    • dining.png - 32x32px (use online converter to create .ico from PNG)
    • restaurant-icon-16.png - 16x16px
    • restaurant-icon-32.png - 32x32px
    • restaurant-icon-180.png - 180x180px (Apple touch icon)
    • restaurant-icon-192.png - 192x192px (Android)
    • restaurant-icon-512.png - 512x512px (Large displays)
  3. Replace the existing files in the /public folder
  4. Alternative: Use online favicon generators like RealFaviconGenerator
